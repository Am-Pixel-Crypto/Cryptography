# Cryptography
Simple GUI App for encoding and decoding messages using matrices.

## Dependencies
 1. **CMake** along with a building system, i.e **MinGW Makefile**.

## Building
<code>cmake -B build -S . -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Debug</code>
<br>
<code>make -C build</code>

<br>
You should find the executable in
<code>build/crypto</code>
